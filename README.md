# 🎯 Random Quote Generator

A clean and interactive web application that displays random inspirational quotes from famous personalities. Built using HTML, CSS, and JavaScript.

---

## 🌐 Live Demo

[Click here to view the live app]( https://prabha092007.github.io/CodeAlpha_random-quote-generator/)

---

## 📌 Features

- 🔀 Displays a **random quote** every time the app loads
- 🔁 **New Quote** button to fetch a different quote instantly
- 🏷️ **Category Filter** — filter quotes by Motivation, Wisdom, Life, Courage, Success
- ❤️ **Save Favourites** — save your favourite quotes (stored in localStorage)
- 📋 **Copy to Clipboard** — copy any quote with one click
- 🐦 **Share on Twitter** — tweet your favourite quotes directly
- 💬 **Share on WhatsApp** — share quotes with friends on WhatsApp
- 🗑️ **Remove Favourites** — remove saved quotes anytime
- 🔔 **Toast Notifications** — smooth feedback messages for every action
- 📱 **Fully Responsive** — works perfectly on mobile, tablet, and desktop

## 💡 How It Works

1. **On load** — a random quote is picked from the quotes array and displayed
2. **New Quote button** — picks a new random quote, avoiding immediate repeats
3. **Category filter** — filters the quotes pool and picks randomly from that category
4. **Save button** — stores the quote in `localStorage` so it persists across sessions
5. **Share buttons** — opens Twitter/WhatsApp with the quote pre-filled
